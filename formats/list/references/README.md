# List format reference screenshots

Store durable screenshots used to derive this format here, preserving slide order in filenames such as `01-cover.png`, `02-body.png`, and so on.

Record the source URL, account, capture date, slide order, and known performance context in `../guide.md`. If any provenance is unavailable, label it as unknown rather than inferring it.

These files are analysis evidence for the format-copy grammar. They are not renderer assets and must not be reused as production imagery unless separately added to `slides-marketing` for that purpose.
