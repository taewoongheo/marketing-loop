# Learning candidates

This file is a small, durable review inbox for repeated cross-content patterns that are not yet approved for promotion.

Do not add:

- one-off corrections;
- raw feedback transcripts;
- draft history;
- copied performance metrics;
- already-approved rules;
- completed-work logs.

When a candidate is approved, move it to exactly one final owner and remove it here. When disproven, remove it.
