# Message MSG-001 v3: The setup is not the work

**Status:** testing
**Version:** 3

## Target user and situation

A U.S. college student has already decided what to study but is still choosing timer, Screen Time, allowed-app, sound, break, or cycle settings before touching the actual work.

## Current belief

Choosing the focus setup is harmless preparation. If starting still feels difficult afterward, the problem must be a lack of discipline or focus.

## Current behavior

The student rebuilds several familiar setup values at the moment of starting while the assignment, reading, or first problem waits.

## Intended belief shift

The setup is not the work. Repeating familiar setup choices may use attention the student intended to bring to the first real study action.

This is a message hypothesis, not an established scientific claim.

## Core perspective

Decide recurring setup values before the study session. At the moment of starting, run the prepared environment instead of choosing its parts again.

Different kinds of study can use different saved environments; reuse does not require one rigid setup for everything.

## Persuasion mechanism

1. Make the concrete setup choices and the still-untouched work visible as two different activities.
2. Separate configuring how to focus from doing the first part of the chosen work.
3. Show that the familiar values can already be decided inside a reusable environment.
4. Define success as contact with the intended work, not completion of configuration.

## Reader resistance to address

- A focus app might become one more thing to configure.
- A saved setup might be too rigid for different kinds of study.
- The repeated choices are too small to matter.
- The real problem may simply be discipline.

## Eligible situations

- A timer is being chosen while the first page remains untouched.
- Sound is being decided before the assignment is opened.
- Blocked and allowed apps are being adjusted before the first problem begins.
- The same timer, break, and sound setup is rebuilt before recurring sessions.

These are semantic examples of the message, not prescribed copy. The selected format's `copywriting.md` owns wording, hook, progression, tone, and product reveal.

## Why env fits

env saves timer, app-blocking, allowed-app, sound, break, cycle, and related settings as reusable environments. The student can run the relevant prepared environment without rebuilding those values at the starting moment.

## Available evidence

- Verified product capability: setup values can be saved in reusable environments and run again.
- Low-confidence desk-research language describes students intending to study while continuing to use their phones.
- No direct evidence currently establishes how often the target audience rebuilds this combination of settings, experiences the choices as a concentration cost, or recognizes setup as the problem source.

## Do not exaggerate

- Do not say each choice measurably lowers concentration unless evidence is added.
- Do not guarantee that env creates focus, prevents every distraction, or improves grades.
- Do not claim that env solves motivation, chooses the work, or performs it.
- Do not present desk-research language as testimony.
