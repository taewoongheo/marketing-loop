# Materials: denzel

This file owns the pre-approved content material available when creating a `denzel` slideshow. It does not own message strategy, product truth, copywriting rules, renderer structure, or final slide copy.

- Select material from this file instead of introducing ad-hoc internet tips.
- Use only material that supports the selected message and stays within its evidence and claim limits.
- User approval is sufficient for practical material. Material without external provenance must not be presented as verified scientific fact, env-user testimony, or a guaranteed result.
- The quoted language below is approved creative input, not verified verbatim evidence unless provenance is recorded by another authoritative owner.
- Update this file when the approved Denzel content pool changes.

| 뇌가 안 먹음 | `I feel braindead`, `I can't study, focus, think, nothing`, `my brain just refuses to cooperate`, `my brain refuses to focus`, `I read the same thing and nothing goes in` | 내 머리가 고장난 것 같은 느낌. 더 강한 마음가짐이나 더 센 압박으로 해결되지 않는 날에 대한 무력감. | 너무 clinical하게 쓰면 위험하다. 앱이 치료한다고 말하지 않고, `some days you need an easier way in` 같은 낮은 진입 경로로만 연결해야 한다. | 첫 행동을 너무 작게 만들기, 집 주변 한 바퀴 걷기, 공부 시작 전 예열 문제 풀기 |
| 공부 방식 불안 | `how do I study the right way without feeling like I'm wasting time?`, `am I wasting time taking notes?`, `what should I study first?`, `overthinking what to study next`, `perfect study routine` | 공부를 시작하기 전부터 방법이 틀릴까 봐 불안한 상태. 준비, 루틴, 노트법, 타이머를 고르다가 실제 공부에 쓸 집중력이 줄어든다. | 세팅 과정이 집중력을 빼앗는다는 상위 주제와 잘 맞는다. `좋은 방법을 찾는 것`이 아니라 `선택을 줄이는 것`으로 전환할 수 있다. | 공부 전 선택 줄이기, 읽은 뒤 덮고 떠올리기, 스스로 설명하기, Error-focused review / retrieval diagnosis |

Denzel 포맷에서 가장 강하게 건드릴 수 있는 harmful belief 풀이다. | 훅의 두 번째 줄이나 커버 subline에 쓰기 좋다. 다만 본문에서는 이 믿음을 강화하지 말고 부드럽게 낮춰야 한다. | `잃어버린 시간 만회`를 포기하기, 첫 행동을 너무 작게 만들기 |

| task initiation / executive dysfunction 언어 | `I know what to do, but somehow I can't make the link to do it`, `task initiation`, `invisible barrier`, `I can't transition into the task`, `I want to do it but I can't start doing it` | 해야 할 행동을 알고 있어도 행동으로 연결되지 않는 답답함. 설명하기 어렵지만 반복되는 막힘으로 느껴진다. | 언어는 강력하지만 앱에서 직접 치료/ADHD 해결처럼 말하면 안 된다. 마케팅에서는 `transition`, `way in`, `first action`, `visible start` 정도로 낮춰 쓰는 것이 안전하다. | 첫 행동을 너무 작게 만들기, If-then으로 시작 조건 정하기, 공부 시작 전 예열 문제 풀기 |
| 더 똑똑해지고 싶음 | `how to become more intelligent?`, `how do I get smarter?`, `how can I become smarter so I won't struggle in school?`, `I want to become smarter` | 공부를 조금 잘하는 수준이 아니라, 내 머리 자체가 더 좋아졌으면 하는 욕망. 성적 문제 뒤에 있는 identity 욕구다. | 강한 훅 소재지만 `IQ 상승`, `지능 향상 보장`처럼 말하면 과장으로 보일 수 있다. `study in a way your brain can actually use`처럼 낮춰 쓰는 편이 안전하다. | Teach-back / generative explanation, Retrieval practice / active recall, Self-explanation / elaboration |

## Methods

| 내부 원리/방법론 | 행동 기반 설명 | 실제 수행 장면 | 체감 변화 언어 |
| --- | --- | --- | --- |
| Retrieval practice / active recall | 읽은 걸 다시 눈으로 확인하는 대신, 책을 덮고 머리에서 직접 꺼내보는 공부. `기억하려고 더 읽기`가 아니라 `기억나는지 먼저 꺼내보기`에 가깝다. 실제 장면은 한 단락을 읽은 뒤 화면/책을 가리고, 핵심 단어 3개나 방금 이해한 흐름을 빈 종이에 적어보는 것. | 한 단락을 읽고 바로 다음 단락으로 넘어가지 않는다. 책을 덮고, 방금 읽은 걸 내 말로 2-3개만 꺼내본다. | 읽었다는 느낌보다 실제로 남은 게 보인다. 책을 덮었을 때 비는 부분이 바로 다음에 공부할 곳이 된다. |
| Practice testing before rereading | 노트를 다시 읽기 전에, 먼저 빈 종이에 내가 아는 것부터 쓰는 공부. 다시 읽기는 첫 행동이 아니라 채점지가 된다. 실제 장면은 노트를 열기 전에 `이 단원에서 기억나는 것`을 2분 동안 쓰고, 그 다음 빠진 부분만 확인하는 것. | 노트를 다시 읽기 전에 빈 종이에 내가 기억하는 것부터 먼저 쓴다. 그 다음에 노트를 열고 빠진 부분만 확인한다. | 다시 읽는 시간이 줄고, 이미 아는 부분과 빠뜨린 부분이 갈린다. 익숙함을 이해로 착각하는 일이 줄어든다. |
| Spaced practice / distributed practice | 하루에 오래 붙잡는 대신, 같은 내용을 며칠에 걸쳐 다시 만나는 공부. `오늘 끝내기`가 아니라 `내일 다시 꺼낼 수 있게 남겨두기`에 가깝다. 실제 장면은 오늘 개념을 처음 보고, 내일 3분, 3일 뒤 5분처럼 짧게 다시 꺼내보는 것. | 하루에 끝내려 하지 않고, 오늘 조금 보고 내일 다시 꺼내볼 부분을 남긴다. 복습은 많이 하는 게 아니라 여러 번 다시 만나는 쪽으로 잡는다. | 하루에 다 끝내야 한다는 압박이 줄고, 공부가 여러 번 다시 만나는 작은 단위가 된다. 다음날 새로 보는 느낌이 줄어든다. |
| Spaced retrieval / delayed recall | 공부 직후의 정답률로 끝내지 않고, 시간이 지난 뒤 다시 꺼내보는 공부. `방금 봐서 맞힌 것`과 `진짜 남은 것`을 분리한다. 실제 장면은 오늘 맞힌 문제를 내일 해설 없이 다시 풀거나, 어제 외운 내용을 다음날 아침 빈 종이에 다시 써보는 것. | 공부 직후에 맞힌 걸 끝으로 보지 않는다. 다음날, 며칠 뒤에 같은 내용을 보지 않고 다시 꺼내본다. | 방금 맞힌 것과 진짜 남은 것이 분리된다. 복습할 곳이 감이 아니라 실제 빈칸으로 보인다. |
| Interleaving | 같은 유형을 편하게 반복하는 대신, 비슷하지만 다른 문제들을 일부러 섞어서 푸는 공부. `손에 익히기`보다 `어떤 방법을 써야 하는지 고르는 힘`을 훈련한다. 실제 장면은 미분 문제만 20개 푸는 대신, 미분/적분/극한 문제를 섞고 각 문제 앞에서 먼저 유형을 판단하는 것. | 같은 유형을 다 끝낸 뒤 넘어가지 않는다. 헷갈리는 유형들을 조금씩 섞어서, 문제를 보자마자 어떤 방법인지 고르는 연습을 한다. | 문제를 많이 푸는 느낌보다, 문제를 보고 방법을 고르는 감각이 생긴다. 시험에서 유형이 섞여도 덜 멈춘다. |
| Self-explanation / elaboration | 답을 맞히고 넘어가는 대신, 각 단계가 왜 필요한지 내 말로 설명하는 공부. `풀이를 따라가기`가 아니라 `선택의 이유를 말해보기`에 가깝다. 실제 장면은 공식 하나를 쓴 뒤 `왜 지금 이 공식이지?`, `왜 다른 공식은 아니지?`를 한 문장으로 붙이는 것. | 풀이를 보고 넘어가지 않고, 각 단계마다 `왜 이게 맞는지`, `왜 여기서 이 방법을 쓰는지`를 한 문장으로 말한다. | 맞힌 문제도 내 말로 설명하면서 대충 넘긴 부분이 드러난다. 이해한 척하던 지점이 빨리 보인다. |
| Pretesting effect | 정답이나 해설을 보기 전에, 먼저 틀릴 기회를 주는 공부. `설명을 듣고 이해하기` 전에 `내 머리가 어디로 가는지`를 먼저 확인한다. 실제 장면은 새 문제를 보고 30초라도 첫 접근을 써본 뒤, 그 다음 해설을 보는 것. | 정답을 보기 전에 먼저 틀려본다. 완벽히 풀 필요는 없고, 첫 접근이나 예상 답을 먼저 적어본 뒤 해설을 본다. | 정답을 보는 시간이 수동적인 읽기가 아니라 내가 틀린 방향을 고치는 시간이 된다. 해설이 더 잘 붙는다. |
| Worked example analysis | 예제 풀이를 답으로 보지 않고, 누군가의 선택 과정을 보는 공부. `계산 따라가기`보다 `왜 첫 번째 선택이 저거였는지 훔쳐보기`에 가깝다. 실제 장면은 해설 옆에 `첫 단서`, `첫 선택`, `다음에 알아볼 신호`를 따로 적는 것. | 예제를 볼 때 계산만 따라가지 않는다. `첫 번째 선택이 왜 이거였는지`, `다른 방법은 왜 아닌지`, `다음 비슷한 문제에서 무엇을 알아봐야 하는지`를 표시한다. | 예제를 베끼는 느낌이 줄고, 풀이의 첫 선택을 보는 느낌이 생긴다. 비슷한 문제에서 첫 움직임이 덜 막힌다. |
| Teach-back / generative explanation | 개념을 더 길게 정리하는 대신, 누군가에게 말하듯 짧게 줄여보는 공부. `많이 쓰기`가 아니라 `흐릿한 이해를 말로 잡기`에 가깝다. 실제 장면은 개념 하나를 친구에게 설명한다고 생각하고 3문장으로 말하거나 적는 것. | 개념을 친구에게 설명하듯 3문장으로 줄인다. 길게 말할수록 이해한 게 아니라, 핵심을 못 잡고 있는 것일 수 있다. | 긴 노트가 짧은 설명으로 줄어든다. 말을 못 잇는 부분이 다시 봐야 할 부분이 된다. |
| Contrast comparison / discriminative contrast | 헷갈리는 개념을 따로 외우지 않고, 바로 옆에 붙여서 차이를 보는 공부. `안 헷갈릴 때까지 분리하기`가 아니라 `헷갈리는 지점을 정면으로 보기`에 가깝다. 실제 장면은 두 공식이나 두 개념을 표로 나란히 놓고 `언제 A`, `언제 B`, `겉으로 비슷한 함정`을 적는 것. | 헷갈리는 두 개념을 따로 외우지 않는다. 한 줄씩 나란히 놓고 `언제 이걸 쓰는지`, `언제 저걸 쓰는지`, `겉으로 비슷한 단서`를 비교한다. | 헷갈림을 피하는 대신 차이를 바로 보게 된다. 비슷한 개념이 시험에서 같이 나와도 덜 흐려진다. |
| Delayed self-test / metacognitive check | 공부 직후의 자신감을 최종 판단으로 삼지 않는 공부. `방금 봐서 익숙한 느낌`과 `나중에 혼자 꺼낼 수 있는 상태`를 구분한다. 실제 장면은 공부가 끝난 직후 체크하지 않고, 30분 뒤나 다음날 아무 자료 없이 다시 설명하거나 풀어보는 것. | 공부 직후의 `나 이거 아는 듯`을 최종 판단으로 삼지 않는다. 조금 시간이 지난 뒤, 보지 않고 다시 꺼낼 수 있는지로 판단한다. | 공부 직후의 안심을 덜 믿게 된다. 진짜 확인은 시간이 지난 뒤 꺼내보는 순간이라는 감각이 생긴다. |
| Error-focused review / retrieval diagnosis | 전체를 다시 보는 대신, 먼저 못 꺼낸 부분을 표시하는 공부. 복습을 `처음부터 다시 읽기`가 아니라 `빈칸 찾기`로 바꾼다. 실제 장면은 퀴즈나 빈 종이 테스트를 먼저 하고, 틀린 것/못 쓴 것만 별표로 남겨 다음 복습 목록으로 쓰는 것. | 전체를 다시 읽지 않는다. 먼저 테스트하고, 못 꺼낸 부분만 표시한다. 복습은 `다시 보기`가 아니라 `빈칸 찾기`가 된다. | 처음부터 다시 보는 시간이 줄고, 복습할 곳이 선명해진다. 공부가 불안 달래기가 아니라 빈칸 처리에 가까워진다. |
| 첫 행동을 너무 작게 만들기 | `10분만`, `문제 1개만`, `노트 첫 줄만`처럼 뇌가 반박하기 어려운 크기로 시작한다. |  | 시작 전에 나랑 싸우는 시간이 줄어든다. 공부 전체를 시작하는 게 아니라, 하나의 작은 행동으로 들어가는 느낌이 된다. |
| If-then으로 시작 조건 정하기 | `책상에 앉으면 바로 1번 문제를 푼다`, `쉬고 돌아오면 방금 보던 문장부터 읽는다`처럼 상황과 행동을 미리 묶는다. |  | 다시 마음먹을 필요가 줄어든다. 그 순간에 무엇을 할지 새로 정하지 않아도 된다. |
| 폰을 시야 밖에 두기 | 폰을 다른 방, 가방, 책상 밖, 뒤집어 둔 위치처럼 바로 보이지 않는 곳에 둔다. |  | 폰을 참는 데 집중력을 쓰지 않아도 된다. 공부 중에 `안 봐야지`라고 계속 버티는 느낌이 줄어든다. |
| 공부 전 선택 줄이기 | 공부할 때마다 음악, 타이머, 자료, 앱, 조명, 책상 정리를 다시 고르지 않게 미리 정해둔다. |  | 공부 전에 이미 지친 느낌이 줄어든다. 시작 전에 집중력을 써버리지 않는 느낌이 된다. |
| 쉬기 전에 돌아올 지점 남기기 | 다음 문제 번호, 다음 문장, 다음 페이지, 이어서 풀 부분을 열어둔 채 쉰다. |  | 돌아올 때 `뭐 하지?`가 덜 생긴다. 다시 시작이 의지 테스트처럼 느껴지지 않는다. |
| 끝이 있는 휴식하기 | 산책 한 바퀴, 물 마시기, 스트레칭 5분처럼 끝나는 행동으로 쉰다. 무한 스크롤처럼 끝이 흐린 행동은 피한다. |  | 쉬는 게 덜 무섭다. 휴식이 하루 전체를 잡아먹는 느낌이 줄어든다. |
| 집 주변 한 바퀴 걷기 | 폰 없이 5-15분 정도 집 주변이나 건물 주변을 걷는다. |  | 머리가 조금 비워지고 다시 앉기 쉬워진다. 공부를 억지로 밀어붙이는 느낌이 줄어든다. |
| 공부 시작 전 예열 문제 풀기 | 제일 쉬운 문제 1개, 어제 풀던 문제 1개, 이미 아는 개념 1개부터 시작한다. |  | `아예 못 하겠다`에서 `일단 들어갔다`로 바뀐다. 시작이 덜 낯설어진다. |
| 읽은 뒤 덮고 떠올리기 | 한 단락이나 한 개념을 읽은 뒤, 보지 않고 핵심 3개를 적거나 말한다. |  | 공부한 느낌이 아니라 남는 느낌이 생긴다. 이해한 척하던 부분이 드러난다. |
| 여러 날로 나눠 보기 | 하루에 몰아서 끝내려 하지 않고, 짧은 복습을 여러 날로 나눈다. |  | 오늘 다 끝내야 한다는 압박이 줄어든다. 공부가 덜 거대한 덩어리처럼 느껴진다. |
| 스스로 설명하기 | `이게 왜 맞지?`, `이걸 친구에게 설명한다면?`처럼 혼잣말로 개념을 풀어 말한다. |  | 내가 정말 이해한 부분과 대충 넘긴 부분이 분리된다. |
| 비슷한 문제 섞어 풀기 | 같은 유형만 몰아서 풀지 않고, 관련 있지만 다른 유형을 섞어서 푼다. |  | 진짜 구분할 수 있는지 빨리 드러난다. `아는 줄 알았는데`를 시험 전에 발견할 수 있다. |
| Study with me / library 이용 | 조용히 같이 공부하는 사람, study with me 영상, 도서관, 스터디 공간을 이용한다. |  | 혼자 버티는 느낌이 줄어든다. 주변 분위기가 시작을 조금 덜 외롭게 만든다. |
| `잃어버린 시간 만회`를 포기하기 | 3시간 날렸으면 3시간을 만회하려 하지 않고, 25분만 다시 시작한다. |  | 다시 시작이 벌처럼 느껴지지 않는다. 망한 하루를 전부 되돌려야 한다는 압박이 줄어든다. |
